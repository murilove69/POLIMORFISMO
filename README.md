# CONTA

Criar classes que polimorficas que se comuniquem entre si e revelem um email de professor ao aluno e vice e versa.
## Começando

Recrie a hierarquia de classes dessa aula e implemente o método enviar email. Esse método deve receber o corpo da mensagem e acrescentar a ela uma saudação de acordo com a implementação do objeto.

Por exemplo:

Olá Prof.  Alan Turing!

Mensagem ….

Ou

Olá Aluno Joseph!

Mensagem …
### Pré Requisitos

Java - Versão 22.0.2

### Instalação

O projeto é não executavél, pois necessita de uma IDE.
